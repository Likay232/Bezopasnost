Static Application Security Testing

Рукодство по использованию NodeJsScan

NodeJsScan - сканер статического кода, используется для поиска недостатков безопасности в приложениях Node.js

Шаг 1

Необходимо установить Postgres, а также настроить его в core/settings.py

Шаг 2

Установка NodeJsScan из репозитория

GitHub

Шаг 3

Необходимо перейти в каталог NodeJsScan и установить всё необходимое при помощи:
pip3 install -r requirements.txt

Шаг 4

Создаем все необходимые записи в базе данных
Команда: python3 migrate.py

Шаг 5

python3 app app.py
для того чтобы провести тестирование среды

Шаг 6

Для корректной работы устанавливаем gunicorn
Команда: gunicorn - b 0.0.0.0:9090 app:app
